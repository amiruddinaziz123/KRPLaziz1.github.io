    Saya adalah seorang siswa yang sudah bercita cita menjadi seorang hacker dari kecil. Pada saat kecil saya suka menonton film bertemakan hacker. Saya fikir seorang hacker itu keren karena dapat meretas berbagai hal dengan sangat mudah. Dengan bermodalkan laptop atau pc yang mereka gunakan mereka dapat membobol dari jaringan internet, aplikasi, bahkan membobol bank. maka dari itu saat saya masih kecil saya sangat ingin menjadi hacker.

    Pada saat itu sekitar tahun 2020 saat saya berada di bangku SMP, saya mendapatkan peluang untuk mempelajari programmer. Pada saat awal itu saya tertarik programmer karena terlihat seperti hacker yang bekerja di depan komputer dan menuliskan kode kode. Saya pun ikut masuk ke dalam ekstrakulikurer Programming di SMP saya. Waktu pertama disana saya memperkenalkan diri “Perkenalkan nama saya Amiruddin Aziz, saya ingin masuk ke programming karena saya tertarik dengan dunia digital serta di masa depan nanti, keahlian di bidakng digital akan semakin meningkat secara drastis.” Setidakinya itu kata saya.

    Sebelum diterima di ekstarkurikurer programming tentu ada berbagai tes untuk menyeleksi siapa saja yang ikut di dalam ekstra programming. Tes yang saya lakukan misalnya kompetisi untuk menyelesaikan game “Mekorama” dan menyelesaikan game di website yang menggunakan logika seperti pemprogramman untuk menjalankan player. Setelah saya tahu bahwa ada beberapa tes tersebut, saya langsung mengerjakannya.

    Tidak ada teks alternatif tersedia untuk gambar ini Pertama tama saya menyelesaikan misi di game “Mecorama”. Karena saya sangat berantusias dalam menyelesaikan misi tersebut. Saya akhirnya mengerjakan dan menyelesaikan semua level di game tersebut dalam satu malam. Tentu saja hal tersebut membuat saya semakin percaya bahwa saya dapat lolos dalam tes tersebut. Lalu pada keesokan harinya saat saya di sekolah, saya memberitahu kakak kelas saya yang membimbing dalam ekstra tersebut, bahwa saya telah menyelesaikan game “Mecorama”. Bahkan kakak kelas saya pun sempat bilang “padahal susah lho kok bisa ya?”. Kakak kelas saya sempat curiga kalau saya mencari kunci kunci lewat internet. Saya akhirnya menyakinkan bahwa saya sama sekali tidak melihat internet. Semua itu adalah hasil kerja saya seorang.

    Setelah saya menyelesaikan tes pertama saya, saya mulai mengerjakan tes kedua saya yaitu menyelesaikan misi dalam game di internet tadi. Tetapi saya mendapatkan sedikit kendala dalam, bagaimana menerapkan program yang harus saya terapkan, saya sempat tersentak beberapa kali karena tidak tahubagaimana jalan game tersebut. Sepulang sekolah setelah pertemuan ekstra Programming saya bertemu dengan teman satu ekstra. Saya bertanya bagaimana cara kerja game tersebut. Dia pun memberitahu bagaimana game tersebut berjalan, setelah saya diajari ternyata game tersebut berjalan seperti flowchart. Akhirnya saya mengerti cara kerja game tersebut. Jadi sepulang sekolah saya kerjakan lagi di rumah. Saya mempunyai banyak waktu untuk mengerjakan tes tes yang ada sampai minggu depan.

    (Waktu pengumpulan hasil tes)

    Waktu penggumpulan hasil tes pun telah tiba. Saya menyerahkan hasil tes saya selama satu minggu tersebut dalam bentuk file word. Setelah beberapa hari, keputusan peserta yang diterima pun keluar. Saya tidak menyangka, saya dapat di urutan ke 3 dari teman teman saya. Padahal sebenarnya waktu itu saya belum familiar dengan dunia digital. Di pertemuan pertama setelah pengumuman pesrta tersebut saya datang dengan laptop saya yang terbilang rendah. Di pertemuan itu saya diajari cara setup tools apa saja yang akan digunakan untuk mendukung saya dalam mengkoding. Karena saya waktu itu masih bingug cara menggunakan laptop, bahkan saya kesulitan dalam installing editor code yang saya download, saya akhirnya dibantu kakak kelas saya untuk setup segalanya. Itulah awal saya masuk di dunia coding yang saya kira semudah membalikkan telapak tangan.
